# AI-Powered Crime Complaint Registration System
Guardian AI – AI-Powered Crime Complaint Registration System

Guardian AI is an AI-powered chatbot that assists users in understanding and initiating the process of filing crime-related complaints such as theft or loss. It provides structured guidance, draft FIR letters, and important safety steps using Google Gemini AI.

Domain
Artificial Intelligence
Public Safety & Crime Assistance
Conversational AI

Technologies Used
Python
Google Gemini API (models/gemini-2.5-flash)
google-genai
PyQt5 (GUI)

How It Works
User describes an incident (e.g., mobile theft).
Guardian AI understands the intent.
The chatbot generates guidance, a draft FIR, and next steps such as IMEI and SIM blocking.

How to Run
pip install google-genai PyQt5
python guardianai.py


Set your API key as:
set GEMINI_API_KEY=YOUR_API_KEY

Example Query
“I want to file a complaint about my mobile phone being stolen.”
